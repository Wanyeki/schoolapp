Security

Security is one of the major things any software must implement.Schoolapp is well developed with security in mind,this prevents unauthorized parties from accessing and modifying school's crutial data. It has been acheived by incorporating a login panel whereby every user must input his/her credentials before accessing anything.You also can add users and also remove them.

Timetabling

Timetable generation is a very demanding and complex task when done manually. Schoolapp simplifies all that by generating all timetables(teaching,exams,duty_roaster) by click of a button.Thats it. It creates well organized individual timetables for every teacher as well as class timetable for every class, which can be printed at anytime. The timetable can also be accessed in a more simpler way,Schoolapp mobile app.

Easy Results Upload and inserting

Manual filling in most softwares is a very tiresome job for person filling.you have to type out every entry you have. Other than mannual filling ,Schoolapp has additional features which makes filling extremely easy and enjoyous. Simple as selecting a prefilled exel file and the software takes care of the rest(grading,ranking and averaging), With the Schoolapp mobile app,filling is even more simpler,upload your results anywhere at anytime.Access them and modify them anytime with your mobile device.

Perfomance analysis

Other than filling in the results,Schoolapp features perfomance analysis capability ,allowing users to compare visually the perfomance of every class ,together with their perfomance trends across the year. All class means and students results are calculated automatically with almost zero intervention.You can also print any class list containing ranked students for any done exam.

staff and student management

Schoolapp has has every tool you need to manage your students and teachers, from adding editing and deleting to assigning classes. Adding students is even easier. All you need is an exel file, this drastically reduces software migration time. The software is equiped with form 1 enrollment feature with all tools you need including assigning classes automatically based on their kcpe.

Customizing

Customizable and flexibility is crutial incase a school needs to expand in future by adding more classes, change how the software works to obtain a more customized version of the software. Schoolapp has a settings panel on which you can add subjects,classes and also remove them at your will. from the panel you can individualy change how grading is done for every subject. The software comes branded with The schools name and logo.

Reportcard printing

Reportcard generation can be tiresome when wrong softwares are used .Schoolapp is powerful enabling you preview and print any desired Reportcards, which are beautifuly designed.Bulk printing is also supported,select a class and term then press print. You can print and view reportcard for any term,even past terms.Besides printing ,Schoolapp generates reportcards in pdf formats which can be shared or transfered to another computer.

Android app

Desktop software is not enough for this modern world.Shoolapp comes with a mobile app(Android,IOS) which is connected to the schoolapp desktop. It has reminders which use the school's timetable to remind you when you have a class and at what time.No paper works...thats cool. The mobile app allows you to vew result and perfomance not only for your schoolapp but also other schools who are using Schoolapp. Marks Submission just got easier.With the app you can insert and submit at your time remotely.
